# ylands_scene_loader
Example loader for exported Ylands scenes

Supports most basic cube-shapes, slopes, corners, and spikes along with trusty musket ball
> Note: more will be added

## Controls
* Move: `W`, `A`, `S`, `D`, `Q`, `Z`
* Turn: `Arrow Keys`
* Speed Modifier: `Shift` or `Ctrl`
